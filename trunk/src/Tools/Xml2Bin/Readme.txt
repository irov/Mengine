для того чтобы сбилдить хедер файл из protocol.xml нужно запустить HeaderCreationTool.exe *пусть к протокольному файлу* *путь к хедер файлу*
для того чтобы создать бин файл из хмл файла нужно запустить CPPConverterTest *путь к протоколу* *путь к хмл файлу* *путь к выходному бин файлу* (для этого в той же папку должен находится Xml2Bin.dll)

!!ВНИМАНИЕ!!
для того чтобы сбилдить Xml2Bin.dll нужно открыть свойства этого проекта, на вкладке Build Events в поле Post-build event command line прописать ..\ExportDll.exe *(полный!!) путь к получившемуся длл файлу*
